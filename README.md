# Kiara's Treats

A production-ready e-commerce and bakery management platform featuring a public storefront, secure checkout, and a comprehensive backend administrative system.

## Overview
Kiara's Treats modernizes bakery operations by providing a seamless shopping experience for customers and robust inventory/revenue management tools for the business owner.

## Features
- **Public Storefront:** Visual catalog for browsing and cart management.
- **Admin Dashboard:** Secure backend for inventory and sales tracking.
- **Inventory Control:** Real-time stock tracking.
- **Revenue Analytics:** Automated calculation and visualization of sales data.

## Quick Setup
1. **Clone & Install:**
   ```bash
   git clone [https://github.com/FamineDT246/kiaras-treats.git](https://github.com/FamineDT246/kiaras-treats.git)
   cd kiaras-treats
   npm install

2.Configure: Create a .env.local file with your Supabase credentials:
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

3.Launch:
    npm run dev
